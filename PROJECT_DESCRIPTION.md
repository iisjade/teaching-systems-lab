# Project Description

**Teaching Systems Lab** is a portfolio project exploring lightweight web tools for math, computer science, and multilingual classrooms.

The first tool, **Math Language Scaffold Generator**, helps teachers adapt math tasks for multilingual learners while preserving mathematical rigor. Teachers enter a grade level, topic, learning goal, task, and student language context. The tool generates editable supports including vocabulary, sentence frames, Spanish-language scaffolds, teacher launch language, likely misconceptions, and checks for understanding.

This project reflects a classroom-centered design approach: make structure visible, reduce avoidable language barriers, and support students in explaining their mathematical reasoning.

## v0.3 Update

The site also includes an examples gallery with classroom-ready starting points for percent change, proportional relationships, and linear relationships, plus a visible tech/deployment note for portfolio review.


## v0.4 Update

The examples gallery is now interactive. Each example card loads a task into the generator and creates a scaffold automatically, making the prototype easier to demo and more clearly product-oriented.


## v0.5 Update

Added a CCSS-Aligned Rubric Generator as a second tool in the suite. The rubric tool drafts a 1–4 standards-based rubric, student-facing success criteria, teacher look-fors, and a family-friendly explanation for selected middle school math standards.


## v0.6 Update

Added a tool-suite overview that frames the site as a two-tool workflow: first adapt the task for language access, then clarify assessment expectations through a standards-aligned rubric. Also improved navigation, section hierarchy, and responsive polish.


## v0.7 Update

Updated the color system from the original warm beige/green palette to a cleaner slate/indigo visual direction with soft blue-lavender surfaces, stronger contrast, and a more modern portfolio feel.
